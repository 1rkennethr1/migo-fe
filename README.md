# Installation

1. Clone the Repo
  ```
   git clone https://https://github.com/whitewaterjunkie/migo.git
  ```
2. Navigate to the Project Directory
  ```
   cd migo
  ```
3. Install dependencies
  ```
   npm install
  ```
4. Development Mode
  ```
   npm run dev
  ```
